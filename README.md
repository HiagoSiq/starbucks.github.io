# Starbucks Landing Page

Este é um projeto de uma página web simples inspirada na marca Starbucks. A página apresenta uma interface interativa onde o usuário pode trocar imagens de produtos clicando nos thumbnails no menu inferior, além de um botão de inscrição com efeitos visuais.

## Funcionalidades

- **Troca de Imagens**: Clique nos thumbnails para alterar a imagem principal e a cor de fundo da página.
- **Animações**: Efeitos de hover e clique nos elementos, incluindo pulsação na imagem principal e tremor no botão.
- **Responsividade**: Design adaptável para diferentes tamanhos de tela.
- **Botão de Inscrição**: Exibe um alerta e muda o texto do botão após o clique.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilos, animações e layout responsivo.
- **JavaScript**: Funcionalidades interativas, como troca de imagens e manipulação de eventos.

## Como Executar

1. Clone este repositório:
   ```
   git clone https://github.com/HiagoSiq/starbucks.github.io.git
   ```

2. Abra o arquivo `index.html` em seu navegador web.

Alternativamente, se o repositório estiver configurado com GitHub Pages, você pode acessar a versão online em: [https://hiagosiq.github.io/satrbucks.github.io/](https://hiagosiq.github.io/satrbucks.github.io/)
